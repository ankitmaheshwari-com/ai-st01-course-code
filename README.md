# Streamlit Essentials — Code Repository

*From First App to Deployment (AI-ST01)*

This repository contains all starter and finished code examples from the **Streamlit Essentials** book. Each chapter includes:

* **starter** files — the code you will write or complete
* **finished** files — the reference solutions
* **project** folder — a hands-on mini‑project (`app.py` and `app.finished.py`)

---

## 1. Set Up Your Environment

Install all dependencies using:

```bash
pip install -r requirements.txt
```

This installs Streamlit and all packages used across the book.

---

## 2. Run Any Example

### Run a practice file (starter)

```bash
streamlit run chapter-01/starter/practice-01.py
```

### Run the solution

```bash
streamlit run chapter-01/finished/practice-01.py
```

### Run the chapter project

```bash
streamlit run chapter-01/project/app.py
```

### Run the completed project

```bash
streamlit run chapter-01/project/app.finished.py
```
